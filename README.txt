# 🚀 Personal Portfolio Website - Adarsh P

This is a responsive personal portfolio website built using HTML5, CSS3, and JavaScript. It showcases my profile, education, technical skills, personal projects, and contact information. Designed to represent my journey as a Junior AI Data Engineer and Robotics Enthusiast.

## 📄 Sections Overview

- **Home Page**  
  - Greeting, Name, and Role

  - Timeline of my academic background

  - Soft skills and technical skills in AI, Python, Data Science, ML, Web Frameworks, and BI

## 🛠️ Setup Instructions

1. Clone this repository:
   ```bash
   git clone https://github.com/Adarsh-fg/portfolio-website.git
   cd portfolio-website
   ```

2. Open `index.html` in your browser:
   ```bash
   open index.html
   ```

3. Or deploy using GitHub Pages:
   - Go to your repo settings → Pages → Source → Select `main` and `/root`
   - Save and view your hosted site!
